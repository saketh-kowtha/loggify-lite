A professional-grade, lightweight browser logging library engineered for modern web applications. Efficiently captures errors, console outputs, and network requests with minimal performance impact.

Browser-First TypeScript Performance Optimized Production Ready

Installation

npm install loggify-lite

Enterprise Features

  • Comprehensive error tracking with detailed stack traces
  • Advanced console monitoring (log, warn, error) with filtering capabilities
  • Sophisticated network request analysis (Fetch & XHR) with timing metrics
  • Fine-grained sampling rate configuration for optimal performance
  • Persistent IndexedDB storage with automatic log rotation
  • Flexible data export capabilities for analysis
  • Zero production dependencies
  • Minimal bundle size impact (~5KB gzipped)

Professional Implementation

import { initializeLogger } from 'loggify-lite';

// Initialize with enterprise-grade configuration
initializeLogger({
    // Storage Configuration
    maxDBSize: 5 * 1024 * 1024,  // 5MB storage limit
    storeName: 'application-logs',
    dbName: 'loggify-production-db',
    
    // Event Management
    maxEvents: 1000,             // Maximum events per window
    windowMs: 60000,             // 1-minute window
    
    // Intelligent Sampling Rates
    samplingRates: {
        ERROR: 1.0,              // Capture all errors
        CONSOLE_ERROR: 1.0,      // Capture all console errors
        CONSOLE_LOG: 0.1,        // Sample 10% of logs
        CONSOLE_WARN: 0.5,       // Sample 50% of warnings
        FETCH: 0.2               // Sample 20% of network requests
    },
    
    // Feature Controls
    allowConsoleLogs: true,      // Enable console logging
    allowConsoleErrors: true,    // Enable error tracking
    allowConsoleWarnings: true,  // Enable warning capture
    allowNetworkRequests: true,  // Enable request monitoring
    allowErrors: true            // Enable global error handling
});
